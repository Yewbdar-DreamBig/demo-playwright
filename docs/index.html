Java 11 Maven TestNG Playwright GitHub Actions

Project Overview

This project demonstrates automated login testing for the OrangeHRM demo application using Microsoft Playwright with TestNG framework, integrated with GitHub Actions for continuous integration.

Why Playwright?

Fast Execution

Playwright runs faster than Selenium with built-in auto-waiting

Modern Architecture

Built for modern web apps with native support for Shadow DOM, iframes

Auto-Wait

Automatically waits for elements to be actionable

Cross-Browser

Supports Chromium, Firefox, and WebKit out of the box

Getting Started

Prerequisites

  • Java JDK 11 or higher
  • Maven 3.6 or higher
  • Git

Local Setup

1

Clone the Repository

git clone https://github.com/your-username/demo-playwright.git
cd demo-playwright
2

Install Dependencies

mvn clean install -DskipTests
3

Install Playwright Browsers

mvn exec:java -e -D exec.mainClass=com.microsoft.playwright.CLI -D exec.args="install --with-deps chromium"
4

Run Tests

# Run with browser visible
mvn test

# Run in headless mode
mvn test -Dheadless=true
5

View Test Reports

Test reports are generated in: target/surefire-reports/

GitHub Actions Pipeline

Workflow Steps

  1. Checkout code - Gets your repository code
  2. Set up JDK 11 - Installs Java with Maven caching
  3. Install dependencies - Downloads Maven dependencies
  4. Install Playwright browsers - Installs Chromium for testing
  5. Run tests - Executes tests in headless mode
  6. Upload reports - Saves test reports as artifacts
  7. Publish results - Shows test summary in GitHub UI

Triggers

  • Push to main or master branch
  • Pull requests to main or master
  • Manual trigger via GitHub UI
